# Furuta式（回転型）倒立振子の制御設計

Pythonで物理モデルを設計し、実行結果をmatplotlibで可視化できるようにしました。

また、制御モデルをPythonベースで設計し、CopilotでC++にコード生成できます。

生成したC++コードの振る舞いがPythonと一致しているかどうかを、SIL検証できます。

詳細については以下の記事をご参照ください。
